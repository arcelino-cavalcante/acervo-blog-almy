# Tiago 5. Tema: Ajude os desviados voltar.

Publicado em: 2020-11-17T07:13:00.002-03:00

Link original: https://almyalves.blogspot.com/2020/11/tiago-5-tema-ajude-os-desviados-voltarem.html

## Conteudo (HTML original)

<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; O desvio da verdade&nbsp; é&nbsp; uma possibilidade e precisamos estar atentos, tanto em relação a nós, quanto em relação aos irmãos. A natureza pecaminosa, os ensinamentos distorcidos, as más&nbsp; influências, as ações diabólicas,&nbsp; podem colocar em riscos a vida com Deus e nossa forma de viver e morrer.</p><p><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp; &nbsp;</span><span>&nbsp;&nbsp;</span>Diante destas possibilidades, termos alguém que nos ajude e sermos alguém que&nbsp; ajude, será de grande valia. Por isso, saiba oferecer&nbsp; e&nbsp; receber ajuda. Não se isole, nem se ache sábio e forte, a ponto de não aceitar conselhos, exortações, correções e disciplina, pois tais práticas além de manifestarem&nbsp; amor,&nbsp; podem ajudar a não nos perdermos na peregrinação em&nbsp; direção a cidade celestial.&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;Não seja indiferente e nem inoperante, caso perceba que alguém está se desviando da fé, pois os danos podem ser eternos. Deus nos fez corpo, para que possamos oferecer e receber ajuda. Contudo, não force a barra, pois o livre arbítrio é um direito de todos, assim como as consequências do seu bom ou mal uso.&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; Esteja atento, pois as estratégias do diabo para nos desviar sempre são as mesmas:&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 1- Coloca dúvida em nossas mentes;&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 2- Nos&nbsp; afasta de quem verdadeiramente pode nos ajudar;&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 3- Traz pra perto pessoas do mal, disfarçadas de amigas;&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 4- Contamina nossa fé;&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 5- Nos faz achar autossuficientes e sábios aos nossos próprios olhos;&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 6- Nos isola;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 7- Nos abate.&nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</p><p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; <b>Esteja atento, pois basta um pequeno desvio para lhe conduzir a um grande abismo.&nbsp;</b></p><p><br /></p><div class="separator" style="clear: both;
