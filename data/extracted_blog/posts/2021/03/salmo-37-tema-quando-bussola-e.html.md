# Salmo 37. Tema: Quando a bússola é desnecessária

Publicado em: 2021-03-09T04:31:00.003-03:00

Link original: https://almyalves.blogspot.com/2021/03/salmo-37-tema-quando-bussola-e.html

## Conteudo (HTML original)

<p>&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; A bússola é uma objeto criado pelo homem para auxiliá-lo a descobrir uma rota, mas este objeto é desnecessário quando o caminho já está traçado. As vezes, dizemos que queremos fazer a vontade de Deus, mas não sabemos como. No entanto, existem textos bíblicos que nos orientam claramente como proceder, faltando-nos apenas a fé que nos leva à obediência. O Salmo 37 é um destes textos, veja quantas orientações claras, para quem deseja fazer o que agrada a Deus:</p>
